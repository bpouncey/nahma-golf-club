# Nahma Golf Club — Project Instructions

## Repo Overview

This repo is the website project for Nahma Golf Club, a 9-hole public golf course in Nahma, Michigan.

## Active Resources (use these)

- `_seo/` — SEO research, keyword strategy, and content briefs. Reference this when writing copy or planning site structure.
- `MLS Listing/` — Property photos and listing details. Good source for imagery and course description data.
- `photo-scrapes/` — Scraped photos from the web. Reference for visual content.

## Archive (do not use)

- `_archive/` — Contains a previous React/Vite prototype and Relume wireframe work. **Do not reference, read, or build on anything in this folder.** It exists for historical reference only.

## Site Build

The active website build will live at the root or in a dedicated folder (TBD). When in doubt, ask rather than assume.
